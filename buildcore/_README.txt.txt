# Install

1) UNZIP to a folder
2) Double click: Install.cmd
3) Press any button and wait for YTDownloader icon displays on Desktop


# Use

1) Double click on YTDownloader icon
2) Copy youtube link(s) // one per row
3) Select download folder
4, Click: Start Download


# Uninstall

1) Close YTDownloader (if run)
2) Double click: Uninstall.cmd


# Update

As long as everything is working in the application, do not update it.

Updates to the packages may arrive from time to time.
If there is some problem with the application and it is not working properly, you can try updating the packages belonging to the application.
The update tools button will download the latest ffmpeg, ffprobe, ffplay and deno packages and overwrite them in the app folder.

1) To update, start the application and click the Update Tools button.
2) Wait until it downloads everything and says Tools updated succesfully.


Enjoy!

# Telepítés

1) Csomagold ki a ZIP-et egy mappába
2) Dupla klikk: Install.cmd
3) Asztalon megjelenik a YTDownloader ikon
4) Indítsd el, illeszd be a youtube linket, válaszd ki a letöltési mappát, katt a Start Download gombra

# Eltávolítás

1) Dupla klikk: Uninstall.cmd
2) Töröld a mappát ahová kicsomagoltad

# Használat

1) Dupla katt a YTDownloader ikonra
2) Másold be a youtube linket/linkeket // soronként egyet
3) Válasz letöltési mappát
4, Kattints a Start Download gombra

# Frissités

Amíg minden működik az alkalmazásban, addig ne frissítsd.

Időnként érkezhetnek frissítések a csomagokhoz.
Ha valamilyen probléma adódik az alkalmazással és nem működik megfelelően, megrpóbálhatod frissíteni az alkalmazáshoz tartozó csomagokat.
Az update tools gomb letölti a legfrisebb ffmpeg, ffprobe, ffplay és deno csomagokat és felülírja őket az app mappában.

1) Frissítéshez indítsd el az alkalmazást és kattints az Update Tools gombra.
2) Várd meg amíg letölt mindent és kiírja hogy Tools updated succesfully.
